## 📌 Overview

DishGenie is a smart recipe discovery and pantry management platform designed to help users find recipes based on available ingredients, dietary preferences, cooking time, and nutritional requirements.

The application provides a seamless cooking experience through authentication, voice-enabled recipe search, pantry tracking, favorites management, and personalized filtering.

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/sutli19/DishGenie.git
```

Move into the project directory

```bash
cd DishGenie
```

Run using Live Server

Open

```text
login.html
```

or

```text
dashboard.html
```

using VS Code Live Server.
